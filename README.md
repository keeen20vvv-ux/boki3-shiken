# 簿記3級 本試験シミュレーター

日商簿記3級の本試験を **60分・100点満点** でそっくり再現するCBT練習アプリ（単一ページPWA）。

## 特徴
- 第1問（仕訳15問・45点）＋第2問（20点）＋第3問（35点）を本番形式で通し演習
- 60分カウントダウンタイマー・電卓内蔵・大問ナビ
- 自動採点（部分点対応）→ 合否判定・大問別得点・所要時間
- 解答解説（あなたの解答 vs 正解＋計算式付き解説）
- 仕訳特訓モード／間違えた仕訳だけ復習モード
- 学習データはブラウザに保存（オフライン動作）

## 収録模試
- 公開模擬試験
- 直前答練 第2回

## 技術
静的な単一HTML＋JS（バックエンドなし）。GitHub Pagesでホスティング。
